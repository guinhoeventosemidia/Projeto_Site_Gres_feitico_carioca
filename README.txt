README - Site G.R.E.S. Feitiço Carioca (Parallax)

Como usar:
1. Crie esta estrutura de pastas:
   /css/style.css
   /js/script.js
   /assets/images/hero-bg-placeholder.jpg
   /assets/images/historia-bg-placeholder.jpg
   /assets/images/foto1-placeholder.jpg
   /assets/images/foto2-placeholder.jpg
   /assets/images/foto3-placeholder.jpg
   /assets/images/logo-placeholder.png
   index.html

2. Substitua os arquivos de imagens pelos seus (mesmas dimensões aproximadas).
3. Abra index.html no navegador ou envie para o servidor.
4. O formulário está configurado para enviar para gresfeiticodorio@gmail.com via FormSubmit. No primeiro envio, confirme seu e-mail seguindo as instruções que o FormSubmit irá mandar.
5. Para ajustar cores, abra css/style.css e edite :root.

Se quiser que eu gere e envie o ZIP por aqui (quando a ferramenta permitir), eu faço — por enquanto cole os arquivos conforme acima.
